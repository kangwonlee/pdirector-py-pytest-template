# Prompt

## Task Description / 과제 설명

Write a function that adds two numbers and returns the result.

## Input Specification / 입력 사양

Two numeric arguments (int or float).

## Output Specification / 출력 사양

Returns the sum of the two arguments.

## Constraints / 제약 조건

None.

## Examples / 예시

add(2, 3) returns 5
